Projects

CS61Classify - A Hand-Written Number Classifier
RISC-V Assembly - May 2020

Implemented an Artificial Neural Network to identify numbers from the MNIST dataset. It can estimate what number was written in a small image with high precision. The challenge of this project was doing it all in RISC-V assembly. Overall the project was very insightful because it showed me how much we take what high-level languages do for granted.

John Conway's Game of Life
C - September 2020

Built a game of life simulator in C! This was my first major project built in the C language. The simulator takes photos and runs 24 separate game of life instances for each of its 24 color channels. This project was really fun because testing it always led to trippy art and I finally got the hang of C pointers. Also, I was able to make my own name generator in the simulator, which was nice.

PhotoText
Flutter, Dart - August 2020

I built an entire mobile app from scratch in Flutter that converts photos into ASCII art. This was a really fun project because I had very minimal app development in the past, so taking the reigns over an entire project taught me a lot of valuable information. The app let's people choose photos from their Google Photos account, which was my first introduction to an API interface. Saving the text art uses a small JSON database.

Gitlet - A Miniature Version Control System
Java - May 2020

I programmed a standalone implementation of Git entirely from scratch in Java. It includes Git functions such as add, rm, branch, merge, pull, push, log, and checkout. All in all, the project was roughly 1,500 lines of code and includes comprehensive unit and integration tests which is one of the largest I have ever done by myself. It taught me how necessary it is to plan everything well in advance and to make an abundant amount of test cases before moving on. I learned a lot about code organization, file I/O, and the inner workings of Git.

Lines of Action
Java - April 2020

Implemented the board game “Lines of Action” in Java, complete with GUI interface, manual, and an AI the user can play against. The AI implements a minimax decision-making algorithm, optimized with alpha-beta pruning to make the “best” move. This project was tedious because making the correct heuristics for the AI to properly function was extremely complicated and making GUI elements in Java. However, making an AI that can I can hardly beat anymore was an achievement in itself.
